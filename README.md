  カードランク：
    T,J,Q,K,Aの順に強くなります。
  ゲームの流れ：
    プレイヤーは　Player0, Player1　の２人
    各プレイヤーは５種類の内からランダムなカードを１枚受け取る
    プレイヤーは順番にパス・フォールド（p）またはベット・コール（b）どちらかのアクションを選択
    3ラウンド制で、ラウンド開始時最初のアクションはプレイヤー0から始まり次にプレイヤー1の順
    ラウンドは両プレイヤーのパス（pp）、ベットにコール（bb）、ベットにフォールド（bp）で終了し、特にベットにフォールドした場合はゲームそのものが終了する
  アクションと勝敗の決定：
    パス・フォールド（p）チップを賭けず相手に手番を渡す。あるいは、試合を放棄しポットを手放す
    ベット・コール（b）チップを賭ける。あるいは相手の賭けに乗る
    ゲームはショーダウンまたは一方のプレイヤーのフォールドにより勝敗が決まります。
